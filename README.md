# Project3DHouse

## Teclas para movimentar a casa

   - `a`: move para esquerda 
   - `d`: move para direita
   - `w`: move para frente
   - `s`: move para trás
   - `W`: move para cima
   - `S`: move para baixo
   - `+`: aumenta o tamanho
   - `-`: diminui o tamanho
   - `↑`: gira para cima
   - `↓`: gira para baixo
   - `←`: gira para esquerda
   - `→`: gira para direita
   - `SHIFT + ←`: rotaciona para esquerda
   - `SHIFT + →`: rotaciona para direita
   - `ESC`: exit / fecha o programa

## Teclas para controlar a fonte de luz
    
    - `l ou L`: liga/desliga a luz 
    - `h`: move para esquerda 
    - `k`: move para direita 
    - `u`: move para frente 
    - `j`: move para trás 
    - `y`: move para cima 
    - `i`: move para baixo
